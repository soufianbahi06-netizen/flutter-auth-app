# Flutter Auth App 🚀

تطبيق Flutter متقدم للمصادقة والتسجيل مع دعم Google Sign-In و Claude AI.

## الميزات ✨

- ✅ تسجيل دخول بالبريد الإلكتروني
- ✅ إنشاء حساب جديد
- ✅ تسجيل دخول بـ Google
- ✅ تكامل Claude AI
- ✅ واجهة مستخدم عصرية

## المتطلبات 📋

- Flutter 3.0.0 أو أحدث
- Dart 3.0.0 أو أحدث
- Firebase Project
- Google Cloud Console Access
- Claude API Key

## التثبيت 🔧

```bash
git clone https://github.com/soufianbahi06-netizen/flutter-auth-app.git
cd flutter-auth-app
flutter pub get
```

## التشغيل 🎮

```bash
flutter run
```
